Как добавить новую лекцию

Посмотреть в файле lectures.json номер последней лекции.
Новая лекция должна идти под номером +1 (подставить вместо ХХХ).

Загрузить картинку в папку images под именем:
lec-ХХХ.jpg

Заполнить шаблон (не трогать запятые!):

, {
  "id": "lec-ХХХ",
  "title": "НАЗВАНИЕ",
  "author": "АВТОР",
  "topic": "ТОПИК",
  "tags": [],
  "description": "ОПИСАНИЕ",
  "thumbnail": "images/lec-ХХХ.jpg",
  "duration": "",
  "preview": "",
  "status": "Доступ после оплаты"
}


Скопировать весь блок целиком (только сам объект, без разделителей)
и вставить вот сюда: https://codebeautify.org/json-fixer

Нажать Fix JSON.

Скопировать исправленный JSON и вставить в lectures.json
сразу после последней лекции, но до закрывающих символов:

]
}


Сделать commit.
