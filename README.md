# lectorium
Babel's lectorium
1. Посмотреть в файле lectures.json номер последней лекции. Новая лекция будет идти под номером +1 (подставить вместо ХХХ)
2. Картинку залить в папку images под названием lec-ХХХ.jpg
3. Заполнить вот эти данные (не трогать запятые!):
   ===============
  ,
    {
      "id": "lec-ХХХ",
      "title": "НАЗВАНИЕ",
      "author": "АВТОР",
      "topic": "ТОПИК",
      "tags": [],
      "description": "ОПИСАНИЕ",
      "thumbnail": "images/lec-ХХХ.jpg",
      "duration": "",
      "preview": "",
      "status": "Доступ после оплаты"
    }
    ===============
  4. Скопировать все от и до =============== (но без них), и залить вот на этот сайт: https://codebeautify.org/json-fixer
  5. Нажать кнопку Fix JSON
  6. Скопировать получившееся справа в файл lectures.json после последней лекции, но до вот этих двух знаков:   ] }
  7. Нажать коммит.
